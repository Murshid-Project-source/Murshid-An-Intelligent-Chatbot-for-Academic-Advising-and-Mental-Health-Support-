# Murshid-An-Intelligent-Chatbot-for-Academic-Advising-and-Mental-Health-Support-
Many students face academic difficulties and psychological stress throughout their education. For example, students may lack knowledge of university regulations or a safe space to express their feelings or seek guidance when facing academic problems. To address this issue, the project proposes developing an intelligent application  in Arabic, 
powered by artificial intelligence capabilities, that provides comprehensive academic and 
psychological support. 

#  Murshid: AI-Powered Academic Chatbot 
> **Note:** The source code of this project is kept in a Private Repository for intellectual property protection. .

## 🌟 Project Overview
Murshid is an intelligent chatbot designed to support university students by providing accurate academic Advising and Mental Health Support. 

## 🛠️ Tech Stack & Architecture
- **Frontend / Mobile App & Web page:** Flutter & Dart (Cross-platform)
- **Backend / Cloud Services:** Firebase / Supabase
- **Artificial Intelligence:** Customized Large Language Models utilizing **Llama 3** .

## ✨ Key Features
- **Smart Academic Guidance:** Quick answers regarding university regulations, schedules, and student services.
- **Mental Health Support:** Provides a safe, secure, and completely confidential space for students to express their academic anxiety.
- **Advanced AI Integration:** Retrieval-Augmented Generation (RAG) framework for accurate information retrieval.
- **User-Friendly Interface:** Clean, smooth, and modern mobile screens tailored for students.
